Démonstration de l'animation Rainbow Cursor

Instructions :

  • Déplacez votre souris dans la zone ci-dessous pour voir l'animation rainbow
  • L'animation suit le mouvement de la souris avec un effet de traînée colorée
  • Les couleurs changent automatiquement avec le temps
  • L'animation a un effet de pulsation dynamique

Section Home Projects

Cette section contient l'animation rainbow cursor. Déplacez votre souris dans cette zone pour voir l'effet en action. L'animation utilise un canvas HTML5 pour créer des traînées colorées qui suivent le mouvement de la souris.

L'animation est basée sur le code React original que vous avez fourni, mais adaptée pour fonctionner avec du TypeScript/JavaScript vanilla et ciblée spécifiquement sur cette section.
